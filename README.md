# AL HAYAT Car Services Website

Professional bilingual website for AL HAYAT CAR SERVICES (S.P.S - L.L.C), Ajman, UAE.

## Vercel settings
- Framework: Vite
- Build command: npm run build
- Output directory: dist

## Update contact info
Edit `src/main.jsx`:
- phone
- whatsapp
- addressEN / addressAR

## Local preview
npm install
npm run dev
